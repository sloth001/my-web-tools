# 即刻分享卡片

想拥有即刻时代的分享卡片吗? (*不想*)

填入数据,即刻拥有!

**你可以[web.somelou.xyz](https://web.somelou.xyz/jike/card)查看效果**

## 项目说明

项目使用 [Angular CLI](https://github.com/angular/angular-cli) (version 8.1.0)

本来想做全自动的,但是因为技术水平限制,只能做成填数据的版本了.

用户头像可以在jellow下载后再上传

服务器原因,你可能会遇到加载缓慢的情况,可以尝试刷新.
